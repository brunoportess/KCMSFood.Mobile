# KCMSFood

Projeto de exemplo pra mostrar conhecimento básico em xamarin
O que foi usado no desenvolvimento:
- Padrão MVVM (sem framework)
- SwipeView
- SQLite
- CollectionView
- XamarinForms 4.7
